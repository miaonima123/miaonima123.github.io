ES7 预览

  • 数组
    • arr.includes() 数组是否包含某个东西
    • 数组的 arr.keys(), arr,entries()
    • for … in 遍历数组 下标 key
    • for … of 遍历数组 值 value, 不能用于json
  • 数字
    • 求幂 3**8 代表：3的8次方
  • 字符串
    • padStart()/padEnd() 指定宽度，不够就补空格或指定字符
  • 容忍度
    • [1, 2, 3,] 老版数组最后不能有逗号，新的可以有
    • 函数参数最后多的逗号也可以
  • async await
    • 和 generator yield 类似
    • generator 不可以写成箭头函数， async 可以
